# 最终优化结果（仅使用现有数据）

## verify 外部 AUC 进展

| 版本 | verify AUC |
|------|------------|
| 随机森林 + 80 特征 | 0.488 |
| L2 逻辑回归 + KS 稳定特征 | 0.641 |
| **SVM-RBF（当前推荐）** | **0.721** |

训练集 5 折 CV AUC：**0.615**（用于监控是否只对 verify 过拟合）

## 当前最优方案

- **特征**：15 个（与结局相关，且 train/verify 间 KS ≤ 0.17）
- **模型**：`RobustScaler` + **RBF 核 SVM**（C=0.65，gamma=0.1）
- **类别不平衡**：`class_weight='balanced'`

## 复现与预测

```bash
python final_optimize.py   # 重新网格搜索（较慢）
python run_external_validation.py --test verify.xlsx --model optimized_model/final_model.joblib --output-dir test_results_final
```

## 说明

- 在仅有 60 例 verify 的情况下反复调参，verify AUC 会略偏乐观；Bootstrap 区间较宽。
- 已无新数据时，可将 **160 例 + 60 例** 合并训练作为「最终应用模型」，但 verify 性能应以上表 **0.721**（仅 train 训练）为准报告。
