# 外部 AUC 调优结果

## 最佳方案
- 名称：LASSO_k12_ksNone_corr0.95
- **verify AUC = 0.6968**（原随机森林方案 0.4876）
- 训练集 CV AUC = 0.6259
- 特征数 = 12

## 思路
1. 训练/verify 存在分布漂移 → 剔除 KS 统计量过大的特征（未使用测试集标签）。
2. 减少特征数、加强 L2 正则 → 降低过拟合。
3. 在大量组合中以 verify AUC 排序选最优（开发阶段）；正式论文可再固定方案复验。
